# Principios Simples

## Se sentir confortavel no desconfortável
Não precisa reagir a tudo, não precisa saber tudo

## Cuidar das pessoas - Lideres formam lideres

## Retire o Ego da equação
Uma coisa é você pensar "Isso está acontecendo comigo" outra é "Isso está acontecendo".
Quanto mais emocionalmente envolvido, mais dificil resolver, tem haver com o cortex pre frontal.

## Construa uma experiencial mental, não solta
Reconheça padrões
Especialistas reconhecem padrões mais rápido
Não basta viver as experiências

A qualidade da pratica, importa mais que a quantidade

Repetição, sem reflexão, não é crescimento.

## Agir antes de ter todas as respostas
Resolvedores de problema entendem que uma decisão é feita, com o melhor de informação disponivel que você tem naquele momento. Não com informação perfeita.

Quando você age, mesmo com 70% de certeza, faz vc ir, ajustar, entrar no fluxo.
Clareza só aparece depois do movimento.

Você vai usar a dor para crescer ou para se vitimizar?
