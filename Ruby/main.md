## Array
```
[3, 4, 5].max
[3, 4, 5].min
[3, 4, 5].first
[3, 4, 5].last
[3, 4, 5].reverse

# won't change the array
lotto = [3, 8, 5, 6, 4]
lotto.sort
lotto

# will change the array
lotto = [3, 8, 5, 6, 4]
lotto.sort!
lotto

```
