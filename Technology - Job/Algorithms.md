# Algorithms
Measures to check a good Algorithm: Time and space complexity.

## Time complexity
Measure of how much time is spent as an algorithm grows

## Space Complexity
Measure of how much working storage or extra storage is needed as an algorithm grows

We use Big O notation to represent those, time and complexity.

## Linear Search

```O(n)```

## Binary Search
Looking in isolation, Binary search has a logarithmic time, which is more efficient than the linear search.

But, binary search has a prerequisite of having the inputs sorted.
Combining the operation of sorting the input and searching, linear search will be more efficient.

## Merge Sort
Get the array and break it into subarrays, until we have an array with one element.
